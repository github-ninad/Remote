# Templates

Ready-to-use templates from the [REMOTE playbook](../REMOTE.md). Copy, adapt, make them yours.

| Template | Used in | Description |
|----------|---------|-------------|
| [Async Standup](./async-standup.md) | Phase 2 | Daily async update replacing the standup meeting |
| [Weekly Update](./weekly-update.md) | Phase 3 | IC weekly written update: shipped, in-flight, blocked, highlights |
| [1:1](./one-on-one.md) | Phase 3 | Manager/IC 1:1 focused on growth and blockers, not status |
| [90-Day Role Contract](./90-day-role-contract.md) | Phase 3 | Outcome-based role contract with visibility plan and calibration cadence |
| [Manager Friday Notice](./manager-friday-notice.md) | Phase 3 & 4 | Weekly specific written recognition from manager to each direct report |
| [Demo Friday Slot](./demo-friday-slot.md) | Phase 4 | Signup format for biweekly demo sessions |
| [Personal README](./personal-readme.md) | Phase 4 | Optional "how I work" page for each team member |
| [Quarterly Highlight Reel](./quarterly-highlight-reel.md) | Phase 4 | IC's quarterly public summary of shipped work and learning |
| [ADR](./adr.md) | Phase 2 | Architecture Decision Record for non-trivial technical decisions |
| [30/60/90 Onboarding Plan](./onboarding-30-60-90.md) | Phase 4 | Structured onboarding plan for new hires with measured ramp milestones |
| [Async Decision Doc](./async-decision-doc.md) | Phase 2 | Structured async proposal for decisions that don't need a meeting |
| [Incident Post-Mortem](./incident-postmortem.md) | Phase 5 | Blameless post-mortem with timeline, root cause, and action items |
